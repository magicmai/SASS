## sass 安装

- 安装 [ruby](https://www.ruby-lang.org/zh_cn/downloads/) 环境，Windows 平台，可以使用 [RubyInstaller](https://rubyinstaller.org/)。
- 安装 sass：`gem install sass`
- 更新sass：`gem update sass`
- 卸载sass：`gem uninstall sass`
